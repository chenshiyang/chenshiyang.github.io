LeetCode OJ(2) Add two numbers

发表于 2015-09-24   |   分类于 leetcode   |  

题目如下：

You are given two linked lists representing two non-negative numbers. The digits are stored in reverse order and each of their nodes contain a single digit. Add the two numbers and return it as a linked list.

Input: (2 -> 4 -> 3) + (5 -> 6 -> 4)
Output: 7 -> 0 -> 8

原题链接在这传送门。

大意：

给定两个数，这两个数的每个位分别倒序存放于两个链表中。要求计算这两个数的和，并把和按相同的方式放在链表中返回。例如，上面的例子是 342 + 465 = 807。

思路：

注意两个数是倒序存放在链表中的，这样我们就可以从两个链表头开始，一个元素一个元素地遍历链表。算法很简单，时间复杂度为O(max(m, n))。

注意：

  1. 要考虑两个链表长度不一样时的情况。
  2. 进位要注意，特别是两个链表都遍历完时，如果最后进位不为0，记得新建一个节点加上。
